# OBS에서 오버레이를 설정하는 방법

## Quick start

1. ACT에서, Plugins > OverlayPlugin WSServer > Stream/Local 오버레이 탭으로 이동합니다.
2. 모든 설정을 기본값으로 두고 (`SSL 허용`에 체크하면 안됨!) `시작`을 누릅니다.
3. 밑의 URL 생성기에서 원하는 오버레이를 선택합니다.
4. 생성된 URL을 복사합니다.
5. OBS에서 브라우저 소스를 생성하고, URL을 붙여넣습니다.
